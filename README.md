# Alpha Research

Systematic exploration of rule-based trading strategies.

This repository is a personal quantitative research lab focused on:

- Hypothesis-driven strategy development
- Modular backtesting infrastructure
- Risk-adjusted performance evaluation
- Benchmark comparison (vs Buy & Hold)
- Transparent documentation of findings

---

## Research Philosophy

Markets are not predicted — they are tested.

Process:
1. Form hypothesis
2. Implement strategy logic
3. Backtest with transaction costs
4. Compare against benchmark
5. Evaluate risk metrics
6. Document findings

No curve fitting. No hindsight bias. No cherry picking.

---

## Repository Structure

- data/ → raw & processed market data
- research/ → hypotheses, notebooks, findings
- strategies/ → trading logic
- backtests/ → execution engine
- risk/ → position sizing & controls
- utils/ → helpers (data, metrics)
- reports/ → performance outputs
- content/ → research posts & drafts
